2. vim 设计理念: 全部使用键盘而不用鼠标, 减少手指移动, 目光移动

   1. 命令模式
      1. x 删除当前字符
      2. dd 删除当前行
      3. yy + p 复制当前行
      4. p 粘贴剪切板
      5. hjkl  左, 下, 上, 右
   2. 插入模式
      1. a 光标后插入
      2. i 光标前插入
      3. o 当前行后插入一行
      4. O 当前行前插入一行
      5. cw 替换光标所在位置到一个单词结尾的字符
